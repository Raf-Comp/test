== Description ==

AI Chat Assistant to wtyczka WordPress, która dodaje zaawansowanego asystenta AI opartego na Claude.ai od Anthropic. Wtyczka jest dostępna tylko w panelu administratora i pomaga w codziennych zadaniach związanych z zarządzaniem stroną.

= Główne funkcje =

* Integracja z Claude.ai API
* Inteligentne odpowiedzi na pytania
* Pomoc w zarządzaniu treścią
* Analiza i optymalizacja SEO
* Generowanie treści
* Wsparcie dla wielu języków

= Wymagania =

* WordPress 5.8 lub nowszy
* PHP 7.4 lub nowszy
* Klucz API Claude.ai

== Installation ==

1. Prześlij katalog `ai-chat-assistant` do katalogu `/wp-content/plugins/`
2. Aktywuj wtyczkę w menu 'Wtyczki' w WordPress
3. Przejdź do ustawień wtyczki i wprowadź swój klucz API Claude.ai
4. Rozpocznij korzystanie z asystenta w panelu administratora

== Frequently Asked Questions ==

= Czy potrzebuję konta Claude.ai? =

Tak, potrzebujesz konta Claude.ai i klucza API, aby korzystać z wtyczki.

= Czy wtyczka jest dostępna dla zwykłych użytkowników? =

Nie, wtyczka jest dostępna tylko dla administratorów strony.
